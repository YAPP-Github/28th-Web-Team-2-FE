# 디자인팀 확인 요청 사항

개발하면서 디자인팀에 확인·결정 받아야 했던 것들을 날짜순으로 정리했어요.
각 항목은 "어느 화면 / 무엇을 / 어떻게 해주세요" 형태입니다.

화면 이름: F01 시작화면 · F02 닉네임 입력 · F03 내 설문 · F04 링크 공유 · F05 결과 · F06 친구 설문

---

## 6월 23일 (화)

- [공통 · 폰트] 폰트 원본 파일을 가지고 있지 않아서, 나중에 글자 종류를 바꾸거나 추가하려면 디자이너분 원본 파일이 필요해요.
- [공통 · 색/모서리] 버튼이나 입력칸의 둥근 모서리 값에 정해진 규칙이 없어요. "이 정도 둥글기는 몇 px" 식으로 기준을 정해주시면 코드에도 똑같이 맞출게요. (지금은 임시값으로 넣어둔 상태)
- [공통 · 색] 카카오 버튼 노란색(#FEE500)이 우리 색상 목록에 없어요. 정식 색으로 등록할지 알려주세요.
- [F02·F04] 로고의 위쪽 위치가 화면마다 살짝 어정쩡해요. 디자인 기준값이 딱 떨어지는 숫자가 아니라서요. 정리된 값으로 맞춰도 될지 확인 부탁해요.
- [F04 링크 공유] 카카오 공유 버튼에 카카오 심볼(아이콘)을 넣을지 말지 정해주세요. 지금 디자인은 글자만 있는데, 심볼 이미지는 준비돼 있어요.
- [F03·F06 설문] 설문 화면 맨 위에 들어갈 작은 안내 문구(라벨)가 디자인 1차본엔 없었어요. 넣을지 말지 확인 부탁해요.
- [공통 · 배경] 배경 구름 위치를 첫 화면(F01) 기준으로 모든 화면에 똑같이 적용했어요. 화면마다 위치가 달라야 하면 알려주세요.
- [F05 결과] 화면 아래 고정된 공유 바의 높이를 대략 잡아뒀어요. 정확한 높이를 알려주세요.
- [F05 결과] 복사 버튼 아이콘이 디자인과 달라요. 디자인은 '복사' 아이콘인데 지금은 '링크' 아이콘이 들어가 있어요. 복사 아이콘 이미지를 주시거나 확인 부탁해요.

## 6월 24일 (수)

- [F02·F04] 제목 글자 묶음의 위쪽 위치가 어정쩡한 값이에요. 정리된 값으로 맞춰도 될지 확인 부탁해요. (제목과 본문 사이 간격은 디자인과 같아요)
- [F04 링크 공유] 안내 카드 안의 그림 높이를 임의로 통일해뒀어요. 그림 크기 정리 작업이 진행 중이었는데, 확정되면 알려주세요.
- [F04 링크 공유] 안내 카드의 모서리 둥글기·흐림 효과 값에 기준이 없어서 그냥 디자인 수치 그대로 넣었어요. 정식 규칙으로 정할지 확인 부탁해요.
- [F02 닉네임] 입력칸의 위쪽 위치도 어정쩡한 값이에요. 정리된 값으로 맞춰도 될지 확인 부탁해요.
- [F06 친구 설문] 진입 화면 아래 안내 문구의 여백이 디자인과 어긋나요. 정확한 여백을 알려주세요.
- [F05 결과] '다운받기' 버튼의 이미지와 동작이 아직 안 정해졌어요. 디자인 에셋을 기다리는 중이에요.
- [F02 닉네임] 키보드가 떠 있을 때와 닫혔을 때 버튼 아래 여백(24px / 12px)이 맞는지 확인 부탁해요.
- [F04 링크 공유] 제목 색이 화면마다 달라요. F04 제목은 완전 검정(#000)인데 F01 제목은 진회색이에요. 같은 종류의 제목인데 색이 다르니, 하나로 통일해 주세요.
