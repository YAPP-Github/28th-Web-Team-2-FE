## 0. 제품 경계
- **서비스명/도메인**: **looky / looky.my**.
- **한 줄 정의**: *또래 인식 서비스* — **친구들의 설문으로 "나를 찍은 인생네컷(4컷)"을 만들어준다.**
- **비회원 서비스** — 로그인·가입 없음. 회원 개념이 없다(중요 — 인증·세션·재방문 설계의 전제).
- **주인공별 고유 링크(URL)** 로 운영. 별도 계정·서버 식별 없이 **URL이 곧 그 주인공의 공간**(주인공/설문자 신원 분리 불필요). 주인공 기기엔 **로컬스토리지에 닉네임**을 저장해 그 정도로만 "내 것"을 구분(모바일 주 타겟). 24h 경과 시 설문→결과 **자동 전환**.
- **두 역할** (회원/비회원 구분 아님, 행위 구분):
  - **생성자(주인공)** — 인생네컷을 받을 사람. 닉네임 설정 → 자기 설문 → 링크 생성·공유.
  - **참여자** — 주인공의 링크로 들어와 그 사람에 대한 설문을 풀어주는 사람.

## 1. 비즈니스 & 서비스
- **핵심 가치**: *"친구들이 본 나를 인생네컷으로 발견한다."* 조하리의 창 기반 — 본인이 보는 나 vs 친구가 보는 나의 갭에서 반전·발견이 생긴다.
- **이론 토대** (상세 `기획.md`): Big Five 4성향(개방성·성실성·외향성·우호성, 신경성 제외)은 **문항 설계용 내재화 잣대로만** 사용 — 화면·프롬프트에 점수·성향명 **노출 금지**. 실제 동작 틀은 **조하리의 창**.
- **수익 모델**: 없음 (현 단계 **반응 검증 우선**).
- **타겟**: **10~20대 인스타 사용 여성**, 또는 **친구 관계를 중요시하는 사람**. (확정)
- **유입 채널 (핵심 루프)**: 인스타 **스토리 공유** — 서비스 이미지를 스토리로 올리고 **링크는 클립보드로 전달**. 카카오톡 공유도 경로. → 참여자 유입이 핵심.
- **절대 금지 (형용사 톤 — `기획.md` 4-4)**: 비하·외모 평가·차별적 단어, 상처 줄 직설적 부정어 금지. 부정 신호는 부드럽거나 매력적인 표현으로 환원(예: 내향성 → "혼자만의 시간을 아는"). 허용 톤 = 긍정/중립만.

## 2. 핵심 구조

### 인증·식별
- **없음 (비회원).** 가입·로그인 없음. 별도 PIN·비밀번호도 없음.
- **식별 = 고유 링크(URL) + 로컬스토리지 닉네임.** 주인공 기기에 닉네임을 저장하는 정도로만 "내 것"을 구분(모바일 주 타겟). 서버 측 계정 없음.
- **결과 재열람 = 링크 자체가 열쇠.** 링크를 아는 사람은 누구나 결과를 본다(별도 인증 없음). → **링크 노출 = 결과 공개**임을 전제로 설계.

### 유저 플로우
```
[생성자(주인공)]
 닉네임 설정 → 자기 설문(온보딩 필수) → 링크 생성 → 공유(인스타 스토리/카톡)
   → 24시간 동안 설문 수집(참여자 응답 3건 이상 목표)
   → 24시간 경과 즉시 결과 페이지로 전환 (인생네컷 + 설명문)

[참여자]
 링크 진입 → 주인공에 대한 설문(5지선다 상황극 랜덤 N) → 제출
```
- **자기 설문 선행 필수**: 조하리는 "나 vs 친구" 비교라 본인 응답 없으면 절반만 생성됨. 온보딩에서 1회.
- **본인 설문 "잘 모르겠다" 중립 선택지** 열어둠 — 본인 응답이 흐릿할수록 ②④ 반전이 커짐(의도된 연료).

### 설문 (상세 `기획.md` 4-1, 7)
- **200문항 풀** = 4성향 × 50. **전 문항 5지선다 상황극**(동의 척도 안 씀).
- 사람마다 **랜덤 N개** 출제. 본인·참여자가 **다른 문항을 풀어도 됨**(점수 아닌 형용사 추출 방식). **본인·참여자 모두 N = 8문항.**
- **재설문 없음** (비회원). 본인 설문은 온보딩 1회.
- 문항 성향 태그는 내재화 — 화면·프롬프트 비노출.

### 결과 산출 (상세 `기획.md` 4-2~4-5)
- AI가 본인·친구 답변에서 **형용사 집합** 추출 → 조하리 4칸 배정:
  - ① 공개된 나(본인∩친구) ② 친구만 아는 나 ③ 나만 아는 나 ④ 새롭게 발견될 나
- **인생네컷**: 내용 있는 칸(①②③)만 AI 이미지 생성. **빈 칸(주로 ④)은 미리 그린 고정 이미지(안개·물음표)** + 재참여 유도 메시지.
- **설명문**: 본인·친구 형용사·근거로 "넌 스스로 신중하다 했지만 친구들은 모험가로 봤어" 류 문장 생성.
- **참여 기준**: 24h 동안 응답 **3건** 확보 → 경과 시 **결과 페이지 1개** 자동 생성(한 사이클 = 단일 결과 페이지). 한 사람이 여러 번 응답해도 됨(여러 명인지 한 명 다회인지 **구분 안 함**).

### 결과 미달 처리 (결정됨)
- **응답 3건 미달로 24h 경과 시 → 결과 대신 재시도/재생성 유도**(재공유로 더 모으거나 다시 시도). 결과 네컷은 생성하지 않음.
